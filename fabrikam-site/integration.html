Fabrikam InsightCloud 概要

Fabrikam InsightCloud は、点検データを一元管理するクラウド基盤です。現場の撮影から報告書作成までを効率化します。

撮影データ取込

可視光・サーマル・位置情報（ジオタグ）付きデータをアップロードして集約します。

点検レポート自動生成

撮影データから点検レポートの下書きを自動生成し、報告書作成を省力化します。

AI 解析

AI亀裂検出等でひび割れ抽出を省力化します。最終判断は有資格者による確認が前提です。

経年比較

RTK±2cm（カタログ値）の位置精度を活かし、同一箇所の時系列比較で変状管理を高度化します。

SDK / API による連携

SDK／API により、既存の点検管理システムや基幹システムと連携できます。（FR-21・NFR-25）

  • REST API：データの取得・登録、ジョブ管理などをプログラムから操作。
  • データエクスポート：撮影・解析データを外部システムへ連携。
  • Webhook：処理完了などのイベントを外部システムへ通知し、ワークフローを自動化。

対応データ形式

標準的な画像・動画・点群等のフォーマットでエクスポートでき、既存ツール資産を活かせます。（FR-22）

種別主なフォーマット例
静止画JPEG／TIFF
動画MP4
位置情報付き画像／地図GeoTIFF
点群LAS 等

認証・アクセス制御

API 連携では、API キーおよび OAuth による認証に対応します。利用者認証・ロールベースのアクセス制御（RBAC）と組み合わせ、安全に外部連携を構成できます。詳細な暗号化・監査ログ等の仕組みは セキュリティ を参照してください。

拡張性

将来的な台数・機能拡張に対応します。（NFR-25）

  • 導入台数の増加（初期数台から段階的な拡張）に対応。
  • マルチテナントによる組織・拠点単位の管理。
  • 点検ルートテンプレートの拡充（ファームウェア v1.3：2026年7月提供予定で機能拡張を予定）。
連携・拡張に関する仕様値はカタログ値であり、構成・条件により変動する場合があります。

開発者向けドキュメント

SDK／API の詳細仕様・サンプル・認証設定などの開発者向けドキュメントは、サポート・保守サービス の窓口よりご案内します。連携構成のご相談も承ります。
